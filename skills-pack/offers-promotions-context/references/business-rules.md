# Business Rules

## Landing Page Ownership
Some offer landing pages are backend-driven and some legacy offers are not.

### Backend-driven landing page examples
- Cashback Rewards
- Avibra
- Previ

### Non-backend-driven examples
- adjoe
- AmOne (personal loan)
- auto insurance

Do not assume all offers follow the same landing-page ownership model.

## Entry-Point Behavior
If an offer is opened from the discovery list:
- do not show the landing page
- show the expanded version of the discovery card instead

Landing pages are shown only when the offer is opened from other placements.

## Enrollment Behavior
If the user is already enrolled:
- usually skip the landing page
- route directly to the web view

Do not assume onboarding or value-prop UI is shown for enrolled users.
