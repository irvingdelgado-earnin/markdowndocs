# Entry Points

## Discovery List
Behavior:
- opening an offer from the discovery list does not show the landing page
- it shows the expanded discovery card instead

## Other Placements
Examples:
- home promo card
- promo carousel
- post-cashout
- other non-discovery-list placements

Behavior:
- these placements may show the landing page before the web view when the user is not enrolled

## Enrolled Users
Across placements:
- enrolled users usually skip landing pages
- they go directly to the web view
